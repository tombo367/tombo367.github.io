# Tom Borrett Academic Website

This repository is a plain static website designed to publish directly with GitHub Pages at `tombo367.github.io`.

## Files

- `index.html`: site structure and all content sections
- `styles.css`: visual design and responsive layout
- `script.js`: small enhancements for reveal animations and footer year

## Publish on GitHub Pages

1. Create a GitHub repository named `tombo367.github.io`.
2. Add these files to the root of that repository.
3. Push to the `main` branch.
4. In GitHub, open `Settings` -> `Pages`.
5. Set the source to `Deploy from a branch`.
6. Choose `main` and `/ (root)`.
7. Your site will publish at `https://tombo367.github.io/`.
